### Instructions to Run the Project

1. **Download the `Project_2` folder** to your computer.
2. Open **UiPath Studio**.
3. In UiPath, open the file **`main.xaml`** from the `Project_2` folder.
4. Click **“Open Workflow”** and navigate to the **Flowchart**.
5. Inside the Flowchart, edit the paths for:

   * The **main template file** (`main template.docx`)
   * The **output folder** (`Word_files`)
6. Save your changes and run the workflow.
